2. 넣어야할 기능
- 위의 탭
- 로그인(첫 화면부터 로그인 안 하면 못 넘어가도록 하기), 로그아웃
- 회원 세션 유지
- 관리자용은 만들까 고민(더 알아보고 확실히 정하기)
- 간단한 예약 프로그램
- 걍 가능한 템플스테이와 비슷하게 돌아가도록 하기
- svg 파일을 이용한 지도 검색 기능
- 게시판 기능
- 데이터베이스 연동
- 화면 위에 마우스 올리면 별 보이는 기능 구현(1-1학기 프로젝트 참고)
	: 해당 사진에 값을 전달하는 식으로 구현하기
	: 별의 종류는 더 풍부하게
	: 별 올라올 때 약간 은은하게 천천히 올라오게하기
- 진짜 맨 처음 화면에서 별 찾기 서비스를 할건지 아니면 천문대 사이트를 할 건지 물어보기
	: 별 찾기 서비스는 내가 만든것 바탕 + 위의 만든걸로 적절히 짬뽕하기
	: 천문대 사이트는 따로 게시판처럼 만들기
- 진짜 맨 처음 화면은 내가 만든거에다가 버튼 추가하기
